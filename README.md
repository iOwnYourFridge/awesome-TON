## Tools
### [Playground](https://txtracer.ton.org/play/)
Experiment with TVM assembly and FunC code directly in your browser. Write, test, and debug assembly instructions with real-time execution.

### [Code Explorer](https://txtracer.ton.org/code-explorer/)
Compile FunC or Tolk code to assembly and explore the generated bytecode. Perfect for understanding how your smart contracts work under the hood.

### [TVM Specification](https://txtracer.ton.org/spec/)
Browse the complete TVM instruction reference with detailed descriptions, opcodes, stack effects, and control flow information for every instruction.

### [Emulate](https://txtracer.ton.org/emulate/)
Emulate raw messages on TON blockchain. Send single messages or batch multiple messages together to see the full transaction tree and trace execution flow.
